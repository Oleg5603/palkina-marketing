# Контент-план ВКонтакте — Светлана Палкина
## Июнь–Июль 2026 | Ежедневный пост + Stories (обновлено 2026-06-17)

**Смена темпа:** с 3 постов/неделю на ежедневные. При таком темпе писать каждый
пост вручную не нужно — `svet_bot` генерирует текст по теме из `topics.json`
(расширен под ежедневную ротацию) плюс картинку (Kandinsky/FusionBrain),
Светлана за 1-2 минуты правит и публикует. Структура недель 1-4 ниже задаёт
ключевые темы — в дни между ними берутся темы из `topics.json` без жёсткой
привязки к плану.

## Stories — шаблоны (ежедневно, между постами)

Stories короче и быстрее в производстве — не нужен полноценный текст, 1-2 фразы.

- **Опрос дня:** «Узнаёте себя?» с вариантами да/нет по теме сегодняшнего поста
- **Цитата:** одна мысль из текущего поста крупным шрифтом на фото
- **Анонс:** «Сегодня в посте — про [тема]» со стрелкой на основную ленту
- **Вопрос в стикере:** «Спросите меня что угодно про отношения» (раз в неделю, сбор вопросов для будущих постов)
- **За кадром:** короткое видео/фото с фразой от Светланы, без подготовки текста

*Проверить заранее:* для Stories через API нужно право «stories» у токена сообщества —
сейчас не подтверждено, возможно понадобится включить Stories в настройках
группы VK и/или получить отдельное разрешение. Пока можно публиковать Stories
вручную из приложения VK.

---

## Правила публикации

- **Объём:** 800–1500 символов (оптимально для ВК)
- **Фото:** у каждого поста — сгенерированная картинка по теме (Kandinsky/FusionBrain) либо без фото, если генерация не настроена
- **Время:** посты — ежедневно к 12:00, Stories — вечером
- **Бот Свет:** генерирует текст и фото по теме → Светлана правит → публикует в группу
